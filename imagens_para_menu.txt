# Imagens necessárias para a tela inicial de "Epic Sky Boss Battle"

1. fundo_menu.png
- Descrição: Imagem de fundo para a tela inicial.
- Estilo: Céu noturno épico, com nuvens estilizadas, estrelas e talvez uma aurora ou brilho cósmico. Tons predominantes de azul escuro, roxo e toques de amarelo/dourado para estrelas.
- Elementos sugeridos: Silhuetas de monstros voando ao fundo, sugerindo ameaça, mas de forma sutil. Talvez uma silhueta do chefe colossal ao longe, parcialmente encoberta por nuvens.
- Inspiração: Jogos Flash/minimalistas, mas com um toque de grandiosidade.
- Tamanho recomendado: 600x900 px (preenche toda a tela, sem distorção)

2. logo.png
- Descrição: Logo estilizado do jogo.
- Estilo: Letras grandes, com efeito metálico ou brilhante, remetendo a "épico" e "heróico". Pode ter detalhes de espada, asas ou foguetes integrados ao texto.
- Cores: Amarelo dourado, branco, azul claro, com contorno escuro para contraste.
- Elementos sugeridos: Uma espada atravessando o nome, ou asas/foguetes estilizados nas laterais.
- Tamanho recomendado: 400x120 px (proporção horizontal, destaque no topo do menu)

3. botao_iniciar.png (opcional)
- Descrição: Botão para "Iniciar Jogo".
- Estilo: Botão grande, arredondado, com efeito de brilho ou relevo.
- Cores: Azul claro, branco, amarelo.
- Elementos sugeridos: Ícone de foguete ou seta para cima.
- Tamanho recomendado: 320x80 px (proporção retangular, centralizado abaixo do logo)

4. decoracoes_menu.png (opcional)
- Descrição: Elementos decorativos para enriquecer a tela (ex: pequenas estrelas, partículas, nuvens extras, silhuetas de monstros pequenos).
- Estilo: Minimalista, mas com brilho e cor para dar vida ao fundo.
- Tamanho recomendado: Variável, mas sugere-se múltiplos elementos pequenos (32x32 px até 128x128 px), para espalhar pela tela.

# Observações
- Todas as imagens devem ter fundo transparente (exceto o fundo principal).
- O estilo geral deve remeter a ação, aventura e fantasia, mas com simplicidade visual.
- Consulte o GDD para manter a coerência com o universo do jogo.
- As resoluções sugeridas garantem boa qualidade visual e encaixe perfeito na tela 600x900 px.
